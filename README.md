## Building the ISO

```bash
# Install archiso
sudo pacman -S archiso

# Clone this repo
git clone https://github.com/moonlightOS-Meow/moonlightos-meow
cd moonlightos-meow
git checkout v7.0

# Build
sudo mkarchiso -v -w /tmp/iso -o ./out releng
```
